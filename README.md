# GarageTrack — Dağıtım / Release Deposu

Bu depo yalnızca **dağıtım** amaçlıdır; kaynak kod ayrı bir **private** repoda tutulur.

- Derlenmiş sürümler (self-contained `GarageTrack.exe`) **Releases** sekmesinde yayınlanır.
- Uygulama, en son sürümü buradaki `releases/latest` üzerinden (token gerektirmeden) kontrol eder.

## Kurulum
[Releases](../../releases) sayfasından en son `GarageTrack.exe` dosyasını indirin ve çalıştırın. .NET kurulumu gerekmez (self-contained).
